# project idea: calculator
. use variables to store numbers and result
. use list to create history
. use switch statement to chose operator
. use if statement to check input validation
. use for loop to deal with history
. use while loop to create the main loop
. use functions to perform different tasks
. introduce `if __name__ == '__main__'` statement in this lesson
